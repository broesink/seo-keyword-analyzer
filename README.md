# SEO Keyword Analyzer

A command-line tool that fetches keyword suggestions and
exports them to CSV for further analysis.

## Features
- Fetches related keywords via Google Suggest API
- Filters long-tail keywords (3+ words)
- Exports results to CSV with timestamp

## Installation
pip install requests

## Usage
python main.py
